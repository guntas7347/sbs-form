# Form Builder App

A full-stack form builder and response collection system built with **React (Vite)** and **Firebase**.

---

## Features

- Google authentication
- Create and manage forms
- Dynamic field types:
  - Text
  - Paragraph
  - Multiple choice (MCQ)
  - Checkbox
  - Dropdown
- Public/private forms
- Response submission system
- Response viewing (owner-only access)
- Admin support (read + moderation)
- Soft delete for forms
- Deadline + response control

---

## Tech Stack

### Frontend

- React (Vite)
- TypeScript

### Backend / Services

- Firebase Authentication (Google Sign-In)
- Firestore (database)
- Firebase Security Rules (access control)
